# 代码验证SM2使用不当造成的一些致命缺陷

Author：于海波

## 代码说明
  该项目主要分为两个部分，第一部分为用SM2签名算法对不同消息签名但使用了相同的随机数k时，通过公式推导发现私钥d已经泄露；第二部分为分别用SM2和ECDSA两种签名算法对同一消息签名时，
如果使用了相同的随机数k也会导致私钥d泄露，该项目模拟了这两种情况通过验证确实得到了开始保密的私钥d。
  
## 运行指导
  直接运行整个py文件，观察求出的私钥d是否与最初生成的一致。
